<h3 align="center">weaving intent into action</h3>

<p align="center">
  A minimal coding agent that weaves user intent, tool calls, and model responses into coherent output. ~300 LOC Python. Runs in a long loop: message → thinking → tools → response → repeat.
</p>

<h3 align="center">Quick Start</h3>

```bash
uv run python -m loom.cli run      # start the TUI
uv run python -m loom.cli --help   # see all commands
```
